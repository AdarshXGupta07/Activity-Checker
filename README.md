# Activity Checker

A dark-mode weekly activity checker built with Next.js + Tailwind CSS.

## Features

- Tick off activities for each day of the week.
- Add custom activities with category + optional notes.
- Weekly completion analysis with automatic feedback.
- Visual graphs for daily completion and per-activity breakdown.
- Local persistence with `localStorage`.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.
